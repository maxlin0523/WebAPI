# 預計有以下實作

- [x] API測試: Swagger

- [x] 依賴注入: Unity

- [x] 分層架構: Controller(展示層)、Service(商業邏輯層)、Repository(資料存取層)

- [x] 參數隔離: AutoMapper

- [x] ORM: Dapper、Sql Command使用反射(Reflection)組成

- [ ] 效能監控: NanoProfiler

- [ ] 參數驗證: FluentValidation、DataAnnotation

- [ ] 單元測試: FluentAssertion、NSubstitute、AutoFixture

- [x] 資料庫: SQL Server
